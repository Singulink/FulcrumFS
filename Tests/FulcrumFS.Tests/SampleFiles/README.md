### Hand-built (minimal valid files)

None at this time.

### Synthesized at test runtime

Negative ZIP-family tests (missing required entries, wrong mimetype) are constructed in test code via `System.IO.Compression.ZipArchive` because such conditions cannot be expressed as real Office/EPUB files.
